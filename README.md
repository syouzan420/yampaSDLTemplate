# yampaSDLTemplate

`$ git clone https://github.com/syouzan420/yampaSDLTemplate.git`

で `yampaSDLTemplate` のフォルダーを作成し その中で

`$ stack run`

を實行してください

nixOS(2.28.5)環境下であれば動くと思ひます

ライブラリが不足してエラーが出る場合 `stack.yaml` の一番下に 使用してゐるものがリストの中に入ってゐます  

それらに對應するものをインストールしてみてください(nixOSでない場合 實際のライブラリ名は 若干異なる可能性はあります)
